Scoreboard

!gd today
Shows today's scoreboard for the default server
(alternatives: !gd now/points/score/scoreboard)
!gd yesterday
Shows yesterday's scoreboard for the default server
(alternatives: !gd yday/prev)
!gd today/yesterday [server]
Shows the scoreboard for the given server
E.g. !gd today nl74 will show the scoreboard for nl74
!gd setServer [server]
Set the default server to [server]
E.g. !gd setServer nl74 will change the default server to nl74

Enemy city intelligence

!gd intel
Links to the default intelligence index
!gd userscript
Links to the default intelligence userscript
!gd intel [town_id]
Shows the gathered town intel for [town_id]
!gd intel [town_name]
Shows the gathered town intel for [town_name]
!gd setIndex [index_key]
Set the active intelligence index key

GrepoData Bot also listens to town BBcode.
E.g. posting '[town]143524[/town]' will show the town intel for town 143524.
