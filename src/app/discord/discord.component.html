List of commands


Configuration

/help
Shows a list of all commands
/world [world]
Set the default world for your discord to [world]
e.g. /world en141 will change the default world to en141

Daily scoreboard

/today
Shows today's scoreboard for the default world
/today [world]
Shows today's scoreboard for the given world
E.g. '/today nl74' will show the scoreboard for nl74

/yesterday
Shows yesterday's scoreboard for the default world
/yesterday [world]
Shows yesterday's scoreboard for the given world
E.g. '/yesterday nl74' will show yesterday's scoreboard for nl74

/alliance
Shows today's alliance scoreboard for the default world
/alliance [world]
Shows today's alliance scoreboard for the given world
E.g. '/alliance nl74' will show today's alliance scoreboard for nl74
/alliance [world] [day]
Shows today's alliance scoreboard for the given world
[day] can be 'today' or 'yesterday'
E.g. '/alliance nl74 yesterday' will show yesterday's alliance scoreboard for nl74

Player search

/search [name]
Search for players by name
