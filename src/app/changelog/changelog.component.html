v3.01 18-11-2018

In order to comply with the Grepolis rules going forward, our userscript will only allow manual report uploads

- An upload button will now be added to your inbox/forum reports. Using the button will send the intel to your index

- You can use the in-game settings menu to disable inbox or forum parsing

- Automatic background uploading is no longer featured as it is against the rules of Grepolis

v2.04 12-11-2018

Released version 2. General performance improvements and minor bugfixes.

v2.00 12-11-2018

- Version 2 of the userscript allows you to add reports straight from your report inbox
- Added an in-game settings menu allowing you to choose which reports are indexed and when
- Improved report parser to handle more reports
- Report hashes are generated client side
- Report hashes are shared between index users to avoid duplicate uploads
- Report hashes are stored and retrieved between sessions to keep track of already added reports
