How does it work?


Reports will receive an additional 'Index +' button that will allow you to upload the report to your unique enemy city index.
Click on this button to upload the report to the index.

Our service will then analyse the report to extract enemy intelligence and add it to your index. You and your allies can then browse the collected intel on grepodata.com or in-game as can be seen in the figure on the right.

If you want to browse intel while in-game, you need to install the userscript for that index.

Note: You should install the userscript again for each new index you want to use. You can have multiple userscripts installed at the same time.

Keyboard shortcuts

You can use the '[' and ']' keys on your keyboard to navigate to the previous and next report in your reports inbox.

Is it allowed?

Our script does not automate any gameplay elements, and therefore is in compliance with the overall rules.


The script has official approval on the following servers:
  • nl

Approval on other servers is pending since 2018. Unfortunately, Grepolis is currently not evaluating new scripts due to limitations on available developers. However, using our script will not get you banned. You can find more info about this here.

What type of reports are collected?

Currently our userscript allows you to collect published reports that are posted on your alliance forum or sent to you in a personal message. Reports from your reports inbox can also be indexed without publishing them. Converted reports from other third party tools can not yet be indexed.

Is my index secure?

Your index key is only available to you and anyone you choose to share the link with. If you suspect your index has been leaked you can easily request a new key to once again secure the index by using the red button in the top right corner of your index homepage.

We actively keep track of bots and crawlers to prevent unauthorized users to gain access to your index. Please keep in mind that you will be temporarily banned if you try to find unknown index keys.

I forgot my index key

Click here to receive an email containing a list of all indexes you created.
Only the owner of the index can request this email. If you are not the owner, ask them for the key or create a new index for yourself.

Who should install the userscript?

The more players use your index userscript, the more intel will be collected. You should encourage your alliance members to also use the script in order to maximize the collected intel. If you wish to collect your own intel, you need to create a new private index and install the userscript that comes with it. If you create a new index, you need to install the unique userscript that comes with it.

How do i uninstall the userscript?

You can install multiple userscripts (for different indexes) at the same time; you don't have to uninstall the userscript to install a new script.
If you do want to uninstall an old script, navigate to the Tampermonkey dashboard by clicking on the Tampermonkey icon in the top right, and then clicking on 'Dashboard'. Find the script you want to uninstall in this list and remove it from Tampermonkey.

Can I create multiple indexes?

You can create as many indexes as you wish. Please note that you should make a new index for each world you wish to play in.

Can I upload reports automatically (background uploading)?

No. The automatic background upload has been removed from the script as it is against the rules of Grepolis. Automated functionality is not allowed in userscripts. Using such a script will result in an in-game ban of your account when you are detected.

Can I merge my index with a friendly alliance?

Merging and forking of indexes will be part of a future release.
